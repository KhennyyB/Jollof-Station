# Jollof Station 🍚

**Jollof wey dey bring joy** — The official website for Jollof Station, a premium Nigerian food brand founded by Chef Aisha Murphy in Lagos, 2022.

## About

Jollof Station serves authentic Nigerian party jollof rice — smoky, rich, and satisfying — along with breakfast, pasta, and catering for events. This website showcases the menu, brand story, customer reviews, and provides easy ordering via Chowdeck and WhatsApp.

## Features

- 🎠 Smooth hero carousel with food imagery
- 📋 Full interactive menu with pricing (singles, trays, breakfast, after party)
- 🎥 Video slideshow showcasing food preparation
- ⭐ Customer reviews with ratings
- 📱 Mobile-first responsive design
- 🔝 Scroll-to-top button
- 💬 WhatsApp quick order floating button
- 📧 Contact form with validation
- 🍪 Cookie consent banner
- ⚡ Lazy loading & code splitting for performance

## Tech Stack

- **React 18** + **TypeScript**
- **Vite 5** (with manual chunking)
- **Tailwind CSS v3** + custom design tokens
- **Framer Motion** for animations
- **shadcn/ui** components
- **React Router** for navigation
